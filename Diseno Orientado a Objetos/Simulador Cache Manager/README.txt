1) Composicion del TP:

El trabajo practico se compone de 4 archivos fundamentales y un archivo de ejemplo, y a su vez todos son clave para su composicion. El archivo "cache_PeliculasClass.dat" con los datos inscriptos de la memoria cache se encuentra en la carpeta "output", la cual se genera automaticamente al compilarse el programa desde VSCode, junto con el archivo .exe.

2) Compilar el TP:

El TP fue desarrollador en Windows 10, en el IDE VSCode, y desde el mismo se compilo con extensiones pertenecientes a C++. Lo ideal seria abrirlo desde la carpeta, con todos sus archivos en ella, y correrlo desde el main.cpp. Se opto porque no sea interactivo el programa, y este solo genera el proceso de cargar en memoria, modificar datos y agregar nuevos.
